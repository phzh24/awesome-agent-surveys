# awesome-agent-surveys

A personally curated collection of survey papers on Large Language Model (LLM)-based agents. This repository covers core architectures, reasoning, evaluation methods, and domain-specific applications.

The list was compiled by filtering records from DBLP using the keywords "agent survey" and "agent review", with a cutoff date of **2026/02/15**.
